About Me

Mijn naam is Boi. Ik woon in Nederland. Op deze site kan je een aantal van mijn projecten vinden, op mijn github kan je (bijna) alles zien wat ik heb gemaakt. Ik ben een game ontwikkelaar en maak de meeste projecten in Unity. Ik heb ook enige ervaring met Unreal. Mijn favoriete programmeer taal is C# voornamelijk omdat het de eerste taal is die ik leerde. Maar ik wil wel meer C++ gebruiken want van wat ik hoorde en zag heb je veel meer controle over alles. Ik ben erg geintereseerd in open source, oudere technologie en video spellen. Een aantal van de games die ik speel zijn rimworld, guilty gear en age of empires. In mijn vrije tijd ben ik voornamelijk aan het programmeren of speel ik spellen. Op mijn stage plek zou ik vooral willen leren over programmeren, git en hoe je in een team moet werken.
